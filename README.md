## ✨ 주요 기능

### 🧩 26개 프리미엄 컴포넌트

| 카테고리 | 컴포넌트 |
|---|---|
| **히어로 & 헤더** | Header, Hero A, Hero Gradient, Hero Split |
| **소셜 증거** | 📡 실시간 활동 피드, Logo Carousel, Social Proof, Testimonials |
| **기능 소개** | 🧩 벤토 그리드, Features Grid, Image Gallery |
| **전환 최적화** | 🧮 ROI 계산기, ⚔️ 경쟁사 비교표, 🪜 3단계 프로세스, 📰 카드뉴스, Before/After |
| **가격 & 수익** | 💎 월/연 토글 가격표, Pricing 3-Tier, ⏰ 카운트다운 배너 |
| **CTA & 리드** | 📌 플로팅 CTA, CTA 배너, Newsletter, Contact Form |

## 🤝 기여하기

새 컴포넌트를 만들고 싶으신가요? 3단계만 따라하세요:

1. `COMPONENT_REGISTRY`에 `{ defaultProps, schema, render() }` 추가
2. `index.html` 드롭다운에 `<option>` 추가
3. PR 제출!
